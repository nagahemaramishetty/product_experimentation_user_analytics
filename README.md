# Product Experimentation & User Engagement Analytics

## Problem Statement

Consumer-facing social and marketplace platforms rely heavily on experimentation to improve user engagement and retention.
This project analyzes user interaction data to identify behavioral drivers of matches and conversations, and evaluates
product feature changes using controlled A/B experiments.

## Business Questions

- What user behaviors correlate with higher match rates?
- How does early messaging impact user retention?
- Can product personalization increase engagement?

## Key Metrics

- Match Rate
- Conversation Rate
- Day-7 Retention
- Swipe Intensity
